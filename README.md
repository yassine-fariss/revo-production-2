# Revo Production

A cinematic, premium-feel portfolio and services website built with React, Vite, Tailwind CSS, and Framer Motion.

## Tech Stack
- React 18
- Vite
- Tailwind CSS v4
- Framer Motion
- i18next (Internationalization)
- React Router DOM
- TypeScript

## Running locally

```bash
npm install
npm run dev
```
